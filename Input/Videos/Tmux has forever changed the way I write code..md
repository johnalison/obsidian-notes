# Windows

  prefix == control + b 
 `<prefix>` + c  - to create a new window
  `<prefix>` + n  - to move to the next window
  `<prefix>` + p  - to move to the previous window  


# Panes
`<prefix>` + %  - split pane vertically 
`<prefix>` + " - split pane horizontally 
`<prefix>` + (arrow key) to move among panes
`<prefix>` + z to zoom
